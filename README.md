# ai-research-writer.skill

## English

`ai-research-writer.skill` is an Agent Skill for long-running, high-quality research writing workflows.
It is designed for tasks that need iterative drafting, structured checkpoints, and consistent quality control instead of one-shot generation.

This skill is inspired by Anthropic's research on:

- [Long-running Claude](https://www.anthropic.com/research/long-running-Claude)
- [Vibe Physics](https://www.anthropic.com/research/vibe-physics)

It is especially useful for:

- turning scattered notes into coherent research drafts
- writing section by section with evidence-first reasoning
- running revision loops for logic, clarity, and factual consistency
- producing stable long-form outputs with lower hallucination risk

Recommended workflow:

1. Define goals, audience, constraints, and output format.
2. Create a structured outline.
3. Expand each section with evidence-aligned writing.
4. Revise through multiple passes (logic, facts, style, language).
5. Deliver the final version plus optional summary variants.

## 中文

`ai-research-writer.skill` 是一个面向长流程、高质量科研写作的 Agent Skill。
它适合需要多轮迭代、分阶段检查和稳定质量控制的写作任务，而不是一次性生成。

这个 skill 的设计灵感来自 Anthropic 的两篇研究：

- [Long-running Claude](https://www.anthropic.com/research/long-running-Claude)
- [Vibe Physics](https://www.anthropic.com/research/vibe-physics)

它尤其适用于：

- 将零散笔记整理为结构完整的研究草稿
- 按章节推进、以证据为先进行写作
- 通过多轮修订优化逻辑、表达与事实一致性
- 在长文任务中降低幻觉风险并保持输出稳定

推荐工作流：

1. 明确目标、受众、约束和输出格式。
2. 先产出结构化提纲。
3. 按章节扩写并对齐证据与论点。
4. 多轮修订（逻辑、事实、风格、语言）。
5. 交付终稿，并按需生成摘要版本。
